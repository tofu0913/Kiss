# hmark
Auto BST pet WS

## 1. Load addon
```
//lua l kiss
```
## 2. Start the process
```
//kiss start
```
or anyone send a tell "kiss him" (non case-sensitive)

## 3. Stop the process
```
//kiss stop
```
or anyone send a tell "stop" (non case-sensitive)
## 4. Unload addon
It's good habbit to unload the addon when you're not using it.
```
//lua u kiss
```
